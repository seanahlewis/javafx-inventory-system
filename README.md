# javafx-inventory-system

## Description
A GUI-based inventory management application built using JavaFX. Users can add, view, update, and delete inventory items.

## Features
- Add new items
- View inventory list
- Update item details
- Delete items
- Input validation and error handling

## Technologies
- Java
- JavaFX
- Eclipse

## What I Learned
- Building GUI applications with JavaFX
- Scene switching and layout design
- Event handling
- Input validation and error handling
